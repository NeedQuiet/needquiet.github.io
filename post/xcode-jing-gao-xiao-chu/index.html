Xcode 警告消除

Posted on 2021-12-02，1 min read

维护老旧项目时，由于版本过低，导致大量的warning影响编译速度，以下是忽略警告的方式

步骤

  1. 找到buildtime中，想要忽略的警告类型，并右键

  2. 选中 " Reveal in Log "

  3. 我们可以看到代表开启的flag为 "-Wunreachable-code"

  4. 其中 “-W” 代表开启，关闭则为 “-Wno-” : “-Wno-unreachable-code”

  5. Build Setting -> Other Warning Flags 里 添加 “-Wno-unreachable-code” 即可消除此类警告

Flags

类型 Flag
Code will never be executed -Wno-unreachable-code
过期方法 -Wno-deprecated-declarations
实例化，未使用 -Wno-unused-variable
nonnull警告 -Wno-nullability-completeness
