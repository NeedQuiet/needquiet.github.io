MarkDown简单用法

Posted on 2020-01-10，3 min read

Markdown 是一种轻量级标记语言，学习成本也不需要太多，且一旦熟悉这种语法规则，会有一劳永逸的效果。

1.常用语法

  • 标题
    1~4个#加上空格，代表4个字号的标题

  • 高亮强调：
    就是两个反引号把内容括起来 ``

  • 引用符号： >

    用来突出内容

  • 字体加粗 ：

    **字体加粗**

  • 字体斜体：

    *字体斜体*：

  • 菜单：
    菜单用-、+、*加空格都可以

  • 链接：

    • 格式：[标签名字]（链接地址）
    • 百度
      [百度](https://www.baidu.com)

  • 网址：

    • 常规样式：https://www.jianshu.com/u/97ee21e3f77f
    • 超链接样式： https://www.jianshu.com/u/97ee21e3f77f
      <https://www.jianshu.com/u/97ee21e3f77f>

  • 插入图片

    • 例：
    • 代码
      ![描述](图片地址)
#例如： ![blog头像图片](../images/blog.jpg)

  • 分割线：

    // 就是3个星号
